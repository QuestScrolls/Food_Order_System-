# Food Order System

A simple command‑line program that lets users order meals from different cuisines. Currently supports Italian and Indian menus.

## Features
- Display available meals for a chosen cuisine.
- Case‑insensitive meal lookup.
- Input validation for cuisine type and quantity.
- Clear error messages and order confirmation.

## Requirements
- Python 3.6 or higher (no external libraries needed).

## How to Use
Run the script:
```bash
python food_order.py
